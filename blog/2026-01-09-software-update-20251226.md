Hey all,

As we get closer to Android 15, we're releasing a final update to Android 14, to improve on the existing user experience and fix a few things we missed on the way!

{/* truncate */}

### Apps Crashing Fix
> In the previous update, a bug was introduced that caused crashing issues on popular apps (mostly social platforms lol), this is now resolved.

### Keyboard Inconsistency Fixes
> Some smaller issues using the Minimal Keyboard have been resolved.

### E-Ink Quick Settings System Service
> We've migrated away from the accessibility service for quick settings, improving the out of box experience and reducing the chance of bugs. This will also expand into third party support for modifying brightness values for the display and keyboard.

### Gboard Alt Key Fix
> The update introduces an alt key fix for Gboard users.

### Alt + Shift Lock for Gboard + BB Keyboard
> Users of Gboard and BB keyboard mods will now be able to lock the shift or alt key by double pressing to toggle them.
